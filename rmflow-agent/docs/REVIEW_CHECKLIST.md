# REVIEW_CHECKLIST

Review current changes for:

1. Bugs
2. Security or privacy risks
3. Hallucination risks
4. Missing approval checks
5. Missing source IDs
6. Missing audit logs
7. Bad architecture
8. Weak demo flow
9. Unclear error handling
10. Overbuilt features

## Output Format

- Must fix
- Should fix
- Nice to have
- Demo risk
- Final recommendation
